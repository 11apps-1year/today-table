# TodayTable

TodayTable은 매일 먹은 음식을 기록하고, 그 기록을 바탕으로 건강 상태를 점검하며, 더 나은 식습관과 건강 개선에 활용할 수 있는 플랫폼입니다.

## 목표

- 매일 먹은 음식과 식사 패턴을 쉽고 꾸준하게 기록합니다.
- 기록된 식사 데이터를 건강관리 관점에서 돌아볼 수 있도록 돕습니다.
- 사용자가 자신의 식습관을 이해하고 개선할 수 있는 인사이트를 제공합니다.
- 음식, 식단, 건강 목표에 대해 자연스럽게 이야기할 수 있는 경험을 만듭니다.

## 주요 활용 방향

- 하루 식사 기록 관리
- 식습관 점검 및 회고
- 건강 개선을 위한 식단 변화 추적
- 음식과 영양에 대한 대화형 상담 및 피드백
- 개인별 건강 목표에 맞춘 식사 기록 분석

## 핵심 아이디어

TodayTable은 단순한 식단 기록 앱을 넘어, 사용자가 자신의 음식 기록을 기반으로 건강을 점검하고 개선해 나갈 수 있도록 돕는 일상형 건강관리 플랫폼을 지향합니다.
